# 自助UI平台用户手册

## 简介

自助UI平台是一个创新的用户界面设计工具，它结合了可视化拖拽设计和AI智能生成两种方式，让用户能够快速创建现代化的用户界面。无论您是专业设计师还是普通用户，都可以通过本平台轻松创建出高质量的UI界面。

## 核心功能

### 1. 可视化设计器
通过直观的拖拽操作，您可以从组件库中选择各种UI组件，并将它们放置在设计画布上。支持实时预览和属性编辑，让设计过程更加直观高效。

**使用步骤：**
1. 在左侧组件库中选择需要的组件
2. 将组件拖拽到中央设计区域
3. 点击组件以编辑其属性
4. 实时预览设计效果
5. 完成后导出代码

### 2. AI生成器
通过自然语言描述您的界面需求，AI将自动为您生成相应的UI界面代码。

**使用步骤：**
1. 在文本框中详细描述您想要的界面
2. 点击"生成界面"按钮
3. 等待AI生成结果
4. 预览生成的界面
5. 如需要可进行调整后导出

### 3. 组件库
平台提供丰富的预设组件，涵盖基础组件（如按钮、输入框、选择器、开关、进度条）、数据组件（如表格、表单）、反馈组件（如卡片、模态框）以及导航组件（如导航菜单、轮播图）等各类UI元素。同时支持创建自定义组件，满足特殊需求。

### 4. 主题切换
支持多种UI主题风格，一键切换，满足不同审美需求。

### 5. 源码查看
在组件库页面，您可以查看每个组件在不同前端框架下的实现源码。支持Vue.js、React、Angular和Flutter四种主流框架间的切换查看，帮助您更好地理解和学习组件实现。

### 6. 多框架导出
支持将设计导出为多种前端框架代码，包括Vue、React、Angular、Flutter等。

## 快速开始

### 可视化设计流程
1. 访问可视化设计器页面
2. 从组件库中拖拽组件到画布
3. 配置组件属性
4. 点击导出按钮选择目标框架
5. 下载生成的代码文件

### AI生成流程
1. 访问AI生成器页面
2. 输入详细的界面描述
3. 点击生成按钮
4. 预览生成结果
5. 导出代码

## 常见问题

### Q: 是否需要注册账户？
A: 不需要。本平台无需注册即可直接使用所有功能。

### Q: 生成的代码质量如何？
A: 平台生成的代码遵循各框架的最佳实践，结构清晰，易于维护。

### Q: 支持哪些前端框架？
A: 目前支持Vue、React、Angular、Flutter，后续会持续增加更多框架支持。
